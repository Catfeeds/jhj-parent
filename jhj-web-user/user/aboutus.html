1.公司介绍

叮当到家是针对细分人群提供的个性化家庭服务平台。为孕婴群体、老年群体、有品质生活群体等细分人群提供差异化的家政服务品类。

公司在业内首个实施，以通过“移动平台+主云门店系统+服务团队”三位一体的三角业务模式，为中高端家庭用户量身打造个性化服务，包含家务定制、母婴到家、老年到家，基本涵盖各项家庭需求，是家庭专属的家务助手。

公司对服务人员进行持久的服务素质培养，通过礼仪培训、技能培训等，保障服务人员具有较高的行业服务水准。

与现有家政相比，整体上具有明显差异化优势：

变被动服务为主动服务

变零散服务为包年服务

变单人服务为团队服务

变普通服务为高逼格服务

变普通式服务为高逼格服务
