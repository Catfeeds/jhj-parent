油烟机清洗的好处

1.清洗油烟机可极大提高或恢复电器能力，节省耗电，避免电力浪费，延长电器使用寿命

2.经常清洗油烟机有助于预防疾病和细菌。烹饪时都会释放出一些呈黑色状的油烟，长期 吸入会对人的身体造成危害

3.油烟机建议每半年清洗一次

清洗流程

1.拆除滤网、清洁剂浸泡、彻底去除重油污

2.内部高温蒸汽清洗消毒

3.内部、外部擦拭，油烟机焕然一新
