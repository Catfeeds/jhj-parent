2.卫生间深度清洁包含洗衣机一台（免拆）的清洗

3.服务前2小时（含）收取服务费50%的退改费；服务前1小时内及服务时间开始后不得退改订单。

服务内容

1.油烟机、灶台、微波炉、冰箱等厨房电器，橱柜内外深度清洗，全屋高温消毒杀菌。

3.油烟机灶具等电器的清洗

4.物品摆放，物品归类
