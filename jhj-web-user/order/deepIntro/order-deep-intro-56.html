服务介绍

温馨提示：

1.新居开荒包含新房装修及二手房新装修用户，过年大扫除和租房大扫除请点击房屋大扫除

2.新居开荒按照建筑面积计算，服务开始前请您出示相关的房屋建筑面积证明

3.新居开荒60平米起，不足60平米按60平米收费

4.本服务包含擦玻璃服务，不包含地板打蜡服务

服务内容

新居开荒流程

1.扫除所有垃圾并清走堆积物，扫除天花板、墙壁、家私、玻璃窗台、饰物上的灰尘；

2.清除地面、墙面水泥块、漆块、涂料斑点

3.使用人字梯配合中性清洁剂对天花板、灯饰、开关进行彻底清洁

4.使用玻璃清洁剂配合玻璃套装工具对玻璃窗、门、玻璃饰物进行彻底清洁

5.用浴室清洁剂、高效瓷片清洗剂配合1`2米伸缩杆及涂水器彻底清洗卫生间

6.用多功能吸尘吸水机配合中性清洁剂清洗地面
